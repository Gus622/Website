Network Security

Businesses, like individuals, can be vulnerable to attacks and data breaches, this is especially bad as it is possible that Customer and employee data may have been stolen. One of the major steps to ensure security is to protect the companies network from malware, in order to do so companies firstly install antivirus software (link) that protects the network and deletes and quarantines malicious code and threats. Alongside antivirus software the company should also use a firewall for the network, it creates a buffer between the internal server and external ones in order to prevent malware from infecting the system. Most OSs tend to include a firewall so in most cases it is as simple as switching the provided one on.

They should also limit the employee's abilities on company hardware, such as preventing them from downloading untrustworthy apps and only allowing them to download from the Play store or app store. They will also give staff (and in the case of schools, students) the minimum possible privileges required to perform their jobs, this prevents the users from downloading malware and prevents them from falling victim to phishing and other attacks.

Companies also ensure equipment and software up to date to prevent there being any vulnerabilities in the system, applying updates to software is one of the most important things when it comes to security as it prevents malicious 3rd parties from exploiting vulnerabilities in the software. If a product is no longer supported by its provider then software must be replaced with a modern alternative that won’t be vulnerable to attacks.

Phishing attacks are where scammers send fake emails to people with the goal of gaining access to sensitive information, get people to send money etc. In order to avoid falling victim to these attacks you should follow these Guidelines for both users and Companies.

Companies will normally Have layers of protection that prevent this, firstly they will make sure that users on their network are given appropriate privileges for their job. They are given the lowest possible privileges required to perform their jobs so that if a user does fall victim to a phishing or other attack the damage is minimized. only System admins should be given admin privileges that allow them to install software, make changes that affect other users and create accounts on the network. these privileges should only ever be given to IT professionals, while it restricts employees it ensures the network is secure.

Databases for companies should also always be encrypted as they deal with large amounts of sensitive data, that if it became compromised would result in customers, users and the company itself’s data being stolen.
